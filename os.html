Kompyuter Əməliyyat Sistemləri

1. Giriş

Kompyuter Əməliyyat Sistemi (ƏS) kompyuterin bütün işini idarə edən əsas proqram təminatıdır. ƏS olmadan kompyuter və proqramlar işləyə bilməz. ƏS istifadəçi ilə kompyuter arasında əlaqəni yaradır.

2. Əsas Anlayışlar

  • Əməliyyat Sistemi: Kompyuterin aparat və proqram resurslarını idarə edən sistem proqramı.
  • İnterfeys: İstifadəçi ilə kompyuter arasındakı ünsiyyət forması (məsələn, Windows pəncərələri).
  • Fayl Sistemi: Məlumatların yaddaşda saxlanma qaydası (NTFS, FAT32 və s.).
  • Proses: Yerinə yetirilən proqram.

3. Əməliyyat Sisteminin Funksiyaları

  1. İstifadəçi interfeysi təmin edir (pəncərələr, menyular, komanda sətri).
  2. Yaddaşın idarə olunması — RAM üzərində nəhar işlədiyini tənzimləyir.
  3. Proseslərin idarə edilməsi — bir neçə proqramın eyni anda işləməsini təmin edir.
  4. Fayl sisteminin idarəsi — fayl və qovluqların yaradılması, silinməsi.
  5. Qurğularla əlaqə — klaviatura, siçan, printer və s. ilə işləmə.

4. Əməliyyat Sisteminin Növləri

NövXüsusiyyətNümunə
Masaüstü ƏSİstifadəçi üçün nəzərdə tutulmuşdurWindows, Linux, macOS
Mobil ƏSTelefon və planşetlərdə işləyirAndroid, iOS
Server ƏSŞəbəkə və server işlərini idarə edirWindows Server, Ubuntu Server

5. Praktiki Nümunə

Tapşırıq: Kompyuterinizdə aşağıdakıları yoxlayın:

  1. Hansı ƏS-dən istifadə edirsiniz.
  2. ƏS versiyasını tapın (Məs: Windows 10, Ubuntu 22.04).
  3. Fayl sisteminizin növünü tapın.

Windows üçün: This PC → Disk → Properties → File system

6. Tələbə üçün Praktiki Tapşırıqlar

  1. ƏS-in əsas funksiyalarını qısa cədvəl şəklində qeyd et.
  2. Windows və Linux arasında 3 fərqi yaz.
  3. Komanda sətrində aşağıdakı əmri icra et və nəticəni yaz:
    winver (Windows üçün) | uname -a (Linux üçün)

7. Nəticə

Əməliyyat Sistemi kompyuterin düzgün işləməsi üçün əsas proqramdır. O, istifadəçi ilə aparat arasında körpü yaradır və bütün resursları idarə edir.
